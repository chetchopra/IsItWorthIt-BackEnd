# IsItWorthIt BackEnd

This is a Rails API made for the IsItWorthIt App which can be found at LINK

## Usage 
1. Clone the repo 
2. Run ```bundle install```
3. You will need to create, migrate, and seed your database.
4. Run ```rails s ```

## Built With
- Rails 5.2.3
